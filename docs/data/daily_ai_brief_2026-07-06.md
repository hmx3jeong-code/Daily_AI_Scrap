Adapter는 기존 레이어 사이에 작은 모듈을 추가하며, Prefix Tuning은 입력 시퀀스 앞에 학습 가능한 벡터를 붙인다. Prompt Tuning은 가상 토큰을 최적화하여 모델의 동작을 제어하는 방식이다. Hugging Face의 peft 라이브러리는 이러한 다양한 기법들을 통합 제공하여 개발자 접근성을 높였다. 결과적으로 PEFT는 AI 모델의 민주화를 가속화하고 실무 적용 비용을 낮추는 핵심 역할을 한다. 사용자는 작업 특성과 자원 상황에 맞춰 최적의 PEFT 기법을 선택해 적용할 수 있다.
