- [Length Value Model: 토큰 수준 길이 모델링을 위한 확장 가능한 가치 사전 학습](https://machinelearning.apple.com/research/length-value-model) | Apple Machine Learning Research | 2026-07-20 09:00
  - 요약: Apple Machine Learning Research는 토큰 수준에서 생성 길이를 모델링하는 Length Value Model(LenVM)을 발표했다. 현대의 자기회귀 모델에서 토큰은 계산의 기본 단위이며, 생성 길이는 추론 비용과 추론 성능에 직접적인 영향을 미친다. 기존 방식들은 주로 시퀀스 수준의 거친 모델링에 의존하여 세밀한 길이 제어가 어려웠다. LenVM은 각 디코딩 단계에서 남은 생성 길이를 예측하는 토큰 수준의 프레임워크를 제안한다. 이 모델은 길이 모델링을 가치 추정 문제로 공식화하여 접근한다. 구체적으로 생성되는 각 토큰에 일정한 음수 보상을 할당하는 방식을 사용한다. 이를 통해 모델은 최적의 생성 종료 시점을 더 정확하게 판단할 수 있게 된다. LenVM은 확장 가능한 가치 사전 학습을 통해 다양한 작업에서 효율적인 길이 예측 성능을 보여준다. 이러한 접근법은 불필요한 토큰 생성을 줄여 추론 비용을 절감하는 효과가 있다. 또한 추론 과정에서의 효율성을 높여 전반적인 모델 성능 최적화에 기여한다. 실무적으로는 LLM의 응답 길이를 정밀하게 제어함으로써 사용자 경험을 개선할 수 있다. 결과적으로 LenVM은 비용과 성능 사이의 균형을 맞추는 새로운 방법론을 제시한다.
